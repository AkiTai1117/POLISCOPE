# POLISCOPE — 政治傾向診断

30の質問への回答から、政治的な考え方の傾向を「左 −100〜右 +100」の1軸で可視化するWebアプリケーションです。

## 公開サイト

以下のリンクから、実際に診断をお試しいただけます。PC・スマートフォンのどちらからでも利用できます。

### [POLISCOPEで診断してみる](https://poliscope-ochre.vercel.app/)

## アプリ概要

政治に関する30の質問へ5段階で回答すると、回答内容をもとに政治傾向スコアを算出します。診断結果では、左寄り・中道・右寄りといった傾向を数値とスペクトラムバーで確認できます。

政治に詳しくない方でも気軽に利用できるよう、質問を10問ずつ3ページに分け、進捗を確認しながら回答できる構成にしました。

> [!NOTE]
> 本アプリは簡易的な診断コンテンツです。結果は学術的・専門的な政治思想の判定ではなく、特定の政党・候補者・政治団体とも関係ありません。

## 主な機能

- 全30問の政治傾向診断
- 「強く反対」から「強く賛成」までの5段階回答
- 10問ずつ3ページに分けたステップ形式
- 回答済み割合を示す進捗バー
- ページごとの未回答チェック
- `−100〜+100`の政治傾向スコア
- 左右の傾向を示すスペクトラムバー
- Web Share APIを利用した結果共有
- Web Share API非対応環境でのクリップボードコピー
- PC・タブレット・スマートフォン対応
- キーボード操作とスクリーンリーダーを考慮したUI
- 回答内容をサーバーやブラウザーストレージへ保存しない設計

## 診断の流れ

1. トップページから「診断をはじめる」を選択します。
2. 各質問に対して最も近い回答を1つ選択します。
3. 10問回答するごとに次のページへ進みます。
4. 全30問へ回答すると診断結果が表示されます。
5. 結果は共有機能からほかの人へ送ることができます。

## 採点方法

回答の選択肢を`-2〜+2`の点数へ変換し、設問ごとに設定された左右方向の係数を掛けて合計します。合計値を質問数に応じた最大値で正規化し、`−100〜+100`の整数として表示します。

| スコア | 判定 |
| --- | --- |
| `−100〜−61` | 左寄り |
| `−60〜−21` | やや左寄り |
| `−20〜+20` | 中道 |
| `+21〜+60` | やや右寄り |
| `+61〜+100` | 右寄り |

左右方向の設問数を同数にしているため、すべての質問へ同じ回答を選択した場合は中道になります。

```bash

cd POLISCOPE
npm install
npm run dev
```

## 保守・カスタマイズ

主要な設定をデータ・ロジック・画面・スタイルに分離し、変更の影響範囲が分かりやすい構成にしています。

| 変更内容 | 編集する場所 |
| --- | --- |
| 質問文と左右方向 | `lib/questions.ts`の`QUESTIONS` |
| 1ページの質問数 | `lib/questions.ts`の`QUESTIONS_PER_PAGE` |
| 回答の選択肢と点数 | `lib/questions.ts`の`ANSWER_OPTIONS` |
| スコア計算 | `lib/diagnosis.ts`の`calculateResult` |
| 判定境界と説明文 | `lib/diagnosis.ts` |
| 画面遷移と操作 | `components/diagnosis-app.tsx` |
| 配色・レイアウト | `app/globals.css` |
| ページタイトルと概要 | `app/layout.tsx` |

### 質問を編集する場合

`lib/questions.ts`の`QUESTIONS`を編集します。`direction`は、その設問への賛成がどちらへ加点されるかを表します。

```ts
{
  id: 1,
  text: "所得が高い人ほど、より多くの税を負担するべきだ",
  direction: -1,
}
```

- `direction: -1`：賛成すると左方向へ加点
- `direction: 1`：賛成すると右方向へ加点

質問数を変更するとページ数も自動計算されます。各質問の`id`は重複しない連番にしてください。

## データの取り扱い

回答内容と診断結果は、サーバー、データベース、Cookie、Local Storageへ保存しません。回答と採点は利用者のブラウザ上で完結します。
